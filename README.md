# The Personal Dev 2020 WordPress Theme

> A purple theme with yellow highlights, perfect for a freelancer or solopreneur

# Screenshots

## Custom Title block
This block shows up in large font when it's the first block or column block of a page, and in smaller font when used throughout the rest of the page.

# Installing

The master branch is built and ready to go. You can install this theme by [downloading the zip file](https://github.com/ThePersonalDev/thepersonaldev-wordpress-2020/archive/master.zip) from GitHub or cloning the project into your themes folder with: `git clone https://github.com/ThePersonalDev/thepersonaldev-wordpress-2020`
